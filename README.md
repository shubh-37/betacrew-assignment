```BetaCrew NodeJS assignment```

Steps to run this project:
1. Clone this repo from [Betacrew Assignment](https://github.com/shubh-37/betacrew-assignment)
2. Go into the folder in which this repo is stored. (Make sure the Node version is greater than 16.2.0)
3. Run` node main.js` to start your TCP Server and then run `node main.js` to start your TCP Client.
4. You can find the `downloaded JSON` of orders after running the TCP Client in the `same directory`.
